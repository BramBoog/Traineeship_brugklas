Add file/directory to .gitignore file:
	$ echo <filename/path> >> .gitignore
Remove file/dir from repository without deleting from disk:
	$ git rm --cached <filename/path>
Reset pull:
	$ git reset --hard ORIG_HEAD
Change branch name:
	$ git branch -m <new_branch_name>
Copy repository (remote / local):
	$ git clone <repository_url / path_to_repository>
Set user email (all local reps / only current rep):
	$ git config --global/local user.email <email>
